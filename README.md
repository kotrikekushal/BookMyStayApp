# BookMyStayApp
# Use Case 1 – Application Entry & Welcome Message

## Overview
This use case demonstrates how a Java application starts execution and prints a welcome message to the console. It establishes the entry point for the Hotel Booking Management System.

## Goal
To create a simple Java program that runs from the command line and displays the application name and version information.

## Key Concepts
- **Class** – Defines the structure of the Java application.
- **main() Method** – Entry point of the program where execution begins.
- **static Keyword** – Allows the main method to run without creating an object.
- **Console Output** – `System.out.println()` is used to display text in the console.
- **String Literals** – Text inside double quotes used for output messages.
- **Program Flow** – Demonstrates simple top-to-bottom execution.
